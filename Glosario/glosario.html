Operador de Red
Persona o empresa que gestiona las líneas de comunicación y ofrece servicios de telecomunicaciones al público.
Banda 28
Es la denominación alternativa que recibe la banda del rango de los 700 MHz que se usa para complementar las redes 4G (cuyas frecuencias están en el rango de 1700 y 2100 MHz).
Carrier
En telecomunicaciones, se le llama así al operador telefónico o compañía telefónica. Ejemplos de carriers móviles en México son Telcel (América Móvil), Movistar (Telefónica) y AT&T. También existen carriers virtuales, llamados MVNO.
MVNO - Mobile Virtual Network Operator
Operador de red móvil virtual. Se les llama así a compañías que se montan sobre la infraestructura de otra. Ejemplo: Virgin Mobile, que utiliza infraestructura de Telefónica.
Portabilidad
Servicio que permite que los usuarios puedan cambiar de compañía telefónica llevándose su mismo número, sin perderlo.
SMS - Short Message Service
Protocolo que permite el envío de mensajes de hasta 160 caracteres entre teléfonos celulares.
MMS - Multimedia Message Service
Servicio que permite el envío de mensajes multimedia, como audios, imágenes y videos.
A2P
Mensajes enviados de aplicaciones a personas, como los servicios de mensajería masiva SMS.
RCS - Rich Communication Services
Iniciativa para proveer servicios de comunicación enriquecida entre operadores, basado en multimedia IP.
B2C - Business to consumer
Soluciones o negocios que están dirigidos directamente al consumidor.
B2B - Business to business
Soluciones o negocios dirigidos a otras empresas.
IoT - Internet of Things
La interconexión digital de objetos cotidianos al internet.
CRM - Customer Relationship Management
Procesos y sistemas que permiten a una empresa administrar su relación con el cliente.
API - Interfaz de Programación de Aplicaciones
Conjunto de reglas y protocolos que permiten que las aplicaciones de software se comuniquen entre sí.
SIM - Subscriber Identity Module
Tarjeta con un chip que se inserta en dispositivos móviles para almacenar la información del usuario.
e-SIM
Tarjeta SIM electrónica que permite usar servicios de red móvil sin necesidad de una tarjeta SIM física.
Terminal
Dispositivo que permite a los usuarios conectarse a una red pública de telecomunicaciones.
MT - Mobile Terminated
Mensajes o llamadas recibidos por un dispositivo móvil.
MO - Mobile Originated
Mensajes o llamadas realizadas desde un dispositivo móvil.
IMEI
Código único de 15 dígitos que identifica a cada teléfono móvil.
DN - Directory Number
Número de teléfono asignado por el proveedor local de telefonía.
ICCID - Integrated Circuit Card Identifier
Código único de 18 a 22 dígitos que identifica a cada tarjeta SIM.
IMSI - International Mobile Subscriber Identity
Código único de 15 dígitos que identifica a un usuario de telefonía móvil en redes GSM o UMTS.
APN - Access Point Name
Nombre de la puerta de enlace que conecta una red móvil a Internet. Proporciona los detalles para la conexión del dispositivo.
Cobertura
Área geográfica en la que se puede utilizar un servicio, como la telefonía o el internet.
Network Throughput
El rendimiento de la red es la velocidad a la que se transmiten los datos a través de una red, como Internet. Es una medida de la cantidad de datos que una red puede procesar y entregar en un período de tiempo determinado.
¿Como se mide?
En Bits por segundo (bps), Kilobits por segundo (Kbps), Megabits por segundo (Mbps) y Gigabits por segundo (Gbps).
HSS - Home Subscriber Server
Es una base de datos que almacena información sobre los suscriptores y sus dispositivos. Es un componente clave del Subsistema Multimedia IP (IMS) y de las redes LTE.
IMS - IP Multimedia Subsystem
Arquitectura que ofrece servicios multimedia como voz, video y mensajería de texto a través de redes IP.
VLR - Visitor Location Register
Abreviatura de Registro de Ubicación de Visitantes y es un servidor que se encuentra en las redes celulares. Su función es permitir que los usuarios puedan hacer roaming fuera del área de cobertura de su HLR (Registro de Posición Base).

El VLR es un componente fundamental de la conectividad móvil, ya que facilita el enrutamiento de llamadas y el suministro de servicios. Algunas de sus funciones son: Asignar números de itinerancia, Gestionar usuarios inactivos, Controlar el acceso a servicios específicos, Actuar como enlace entre el MSC y el HLR.
HLR - Home Location Register
También conocida como Registro de Localización de Usuarios, es una base de datos que contiene información sobre los clientes de una red móvil, como sus números de teléfono, derechos de servicio e historial de portabilidad.

Te permite saber si existe un número de móvil, a qué operador de telecomunicaciones corresponde y si está conectado al suscriptor adecuado.
Mobile Country Code - MCC
Es un código numérico de tres dígitos que identifica el país de origen de un número de teléfono. Se utiliza para identificar las redes GSM.
El MCC se usa junto con el código de red móvil (MNC) y la identidad internacional del abonado móvil (IMSI). Esta combinación de códigos permite identificar a un operador de red móvil y a un número de teléfono concreto.
El MCC tiene varias aplicaciones, entre ellas: Gestión de la itinerancia, Localización de la red, Identificación de la región geográfica de un abonado móvil.
Mobile Network Code - MNC
El código de red móvil (MNC) es un número de dos o tres dígitos que identifica a una red móvil terrestre pública (PLMN). El MNC es asignado por el regulador nacional.
El MNC se utiliza en combinación con el código de país móvil (MCC) para identificar a un operador de telefonía móvil. Juntos, forman la Identidad de Red Doméstica (HNI), que permite que los dispositivos móviles se conecten a la red correcta.
Red móvil terrestre pública - PLMN
Es un conjunto de servicios de telecomunicaciones inalámbricas que ofrece un operador de telefonía móvil en un país.
Cada PLMN tiene un código único que se compone del código de país móvil (MCC) y el código de red móvil (MNC).
5G
Quinta generación de redes móviles, que ofrece mayor velocidad, menor latencia y la capacidad de conectar más dispositivos simultáneamente.
Ping
Herramienta de red utilizada para medir el tiempo de respuesta entre un dispositivo y un servidor o equipo remoto.
QoS - Quality of Service
Medida de la calidad del servicio en redes, que asegura la transmisión de datos con prioridades específicas para garantizar el rendimiento de aplicaciones críticas.
Roaming
Capacidad de un usuario móvil para utilizar su servicio de red fuera de su área de cobertura local, generalmente en el extranjero.
Backhaul
Conexión entre una torre celular y la red central de un operador, clave para el funcionamiento de redes móviles.
OTT - Over The Top
Servicios que utilizan la infraestructura de internet para ofrecer contenidos, como Netflix, WhatsApp o YouTube, sin intervención directa de los operadores tradicionales.
Latencia
Tiempo que tarda un paquete de datos en viajar desde el origen hasta el destino en una red.
